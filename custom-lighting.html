Personal Project

Custom Lighting Controllers

As part of a personal hobby, myself and my colleague Alec designed a customizable lighting controller. The goal for this project was to set several lighting "moods" into our rooms that were controlled via home assistant.

Overview

Integrating smart home technology with custom hardware.

Custom Lighting Setup

This project involved designing custom PCB layouts and writing firmware to interface with Home Assistant. We wanted granular control over LED strips to create immersive environments tailored to different activities such as studying, gaming, or relaxing.

The system utilizes ESP32 microcontrollers for Wi-Fi connectivity, allowing for real-time state updates and low-latency control. We also integrated support for WLED, which provided a robust base for addressable LED animations.
